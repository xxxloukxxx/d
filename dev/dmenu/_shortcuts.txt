# Navigation
win + [q-p]                 : switch to workspace [q-p] 
win + shift + [q-p]         : switch active window to workspace [q-p] 
win + tab                   : switch to next workspace
win + [left,right]          : focus to prev/next window
win + [j,k]                 : focus to prev/next window
win + [h,l]                 : resize to active windows
win + v                     : toggle vertical mode
win + shift + enter         : switch main window
win + [period,coma]         : focus to prev/next monitor
win + shift + [period,coma] : switch active window to prev/next monitor

# Misc
win + shift + c             : kill active window 
win + b                     : toggle bar

# Mode
win + shift + b             : floating mode
win + shift + space         : floating mode for active window
win + shift + n             : tiled mode
win + space                 : switch between two last used modes
win + shift + m             : monocle mode

# Apps
win + esc                   : scratchpad
win + enter                 : terminal
win + d                     : menu
win + shift + strl + bs     : shutdown menu
win + shift + ctrl + p      : volume 
win + shift + ctrl + s      : screenshot

# Mouse
win + leftclick             : move window
win + rightclick            : resize window
win + middleclick           : toggle floating/tiled mode
